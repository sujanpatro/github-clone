# GitHub Profile Clone

A high-fidelity React clone of the GitHub profile page. Built as a technical assessment to demonstrate UI/UX implementation and API integration.

## Features

- **Real-time Data**: Fetches user profile and stats via GitHub REST API.
- **Contribution Graph**: Custom heatmap implementation using external contribution data.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Modern UI**: Includes skeleton loaders and GitHub's dark theme styling.

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- React Router & Context API
- Biome for linting and formatting

## Setup

```bash
npm install
npm run dev
```
